/**
 * FOUND-05 / D-14: ESLint must block forbidden imports and forbidden APIs
 * inside `src/engine/**`.
 *
 * We lint a temporary fixture file pretending to live at `src/engine/fixture.ts`
 * by lying about the filename to ESLint. (ESLint resolves config based on filename,
 * not on the actual physical location of the source we hand it.)
 */
